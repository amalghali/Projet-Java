# 🎖️ WORKSHOP - Plateforme Éducative "Warzone"

Bienvenue dans ce projet JavaFX révolutionnaire qui transforme l'apprentissage du code en une expérience immersive et militaire.

## 🚀 Fonctionnalités principales
- **Battle Arena RPG** : Un mini-jeu de combat en temps réel. Déplacez votre soldat et répondez à des questions techniques pour neutraliser les drones ennemis.
- **IA Expert Java** : Un chatbot intelligent capable de vous expliquer les concepts de la JVM, du Polymorphisme et de l'Encapsulation.
- **Admin Dashboard** : Interface complète pour gérer les jeux, les scores et la difficulté via une base de données MySQL.
- **Système de Badges** : Gagnez des médailles **Gold**, **Silver** ou **Bronze** selon vos performances au combat.

## 🛠️ Stack Technique
- **Langage** : Java 21
- **Framework** : JavaFX 22 (UI moderne et dynamique)
- **Base de données** : MySQL (JDBC)
- **Audio** : Gestion immersive via AudioClip (sons militaires 8D)

## 🎮 Comment lancer le projet ?
1. Clonez le dépôt.
2. Assurez-vous que MySQL est lancé (Base de données: `workshop`).
3. Compilez avec Maven : `mvn clean compile`.
4. Lancez la classe `main.AppLauncher`.
